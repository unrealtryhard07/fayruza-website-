📐 IMAGE SIZE GUIDE
Recommended: 400×250 pixels (1.6:1 ratio) — fits perfectly in the menu card
Minimum: 300×200 pixels
Format: JPG or WebP for photos, PNG for graphics
Max size: 2MB per image
How it displays: Images are cropped to fill a 100% × 110px box using object-fit: cover. Wider images work best. Square images will crop the top/bottom slightly. Tall images lose a lot on the sides.
Bulk image matching: Name your files exactly like the product name (e.g. Zinger Burger.jpg, karak tea.png). Case-insensitive matching.
